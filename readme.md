# Elmish

Experimental implementation of the [Elm architecture](https://guide.elm-lang.org/architecture/) in [PReact](https://preactjs.com/) inspired by [Elmish](https://elmish.github.io/).  

Have a look to [Elmish Book](https://zaid-ajaj.github.io/the-elmish-book/#/) for more content about the pattern.

## Commands

Initialize and run project:

```bash
pnpm install
pnpm dev
```

Run tests watch:  

```bash
pnpm vitest
```
